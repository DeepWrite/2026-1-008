# 과제-03 단문 연습에 대한 코멘트

## 코멘트

전반적인 글을 읽어보았을 때 OREO 논증문의 구조를 따르려고 노력한 것이 많이 보였다. 하지만 '이 주장의 첫 번째 구성요소인 ‘개인의 표현의 자유를 최대한 보장해야한다’에 대해 더 자세히 분석해보겠다.'와 같은 문장처럼 필요없는 문장이 글의 가독성을 해쳐 진짜 필요한 부분들의 식별성도 떨어뜨리는 점이 아쉬웠다. 또한 예시를 서술하는 문장의 경우에도 내용 그대로 다 작성하기 보다는 단문의 취지에 맞게 더 요약해서 식별하기 쉽도록 작성하여 가독성을 높이면 더 좋을 것이라고 생각된다. 위 두가지 피드백이 반영된다면 자연스럽게 가독성도 증가하고 OREO 논증문의 취지를 더 잘 활용할 수 있을 것이다.
