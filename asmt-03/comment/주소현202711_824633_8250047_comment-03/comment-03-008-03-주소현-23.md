# 과제-03 단문 연습에 대한 코멘트

- 대상과제: `과제-03 단문 연습`
- 코멘트를 제공하는 학생: `008-03 주소현` 
- 코멘트를 받는 학생: `008-23 구은형` 

---

## 코멘트

현재 제출된 OREO 논증문은 전반적으로 OREO의 구조를 잘 이해하고 작성된 글로 평가된다. 특히, 논제에 선행하는 전제를 자세히 설명하여 근거로 활용한 점이 인상깊다. 또한, 실제 연구 결과를 가져와 구체적 사례(E)로서 활용한 점도 높이 평가된다. 논제 재진술문 또한 도입 부분의 진술문보다 더 구체적으로 제시돼, 단순한 반복을 넘어서는 역할을 수행했다. 다만, 현재의 R과 E에서는 청소년을 보호해야하는 이유에 대해서만 구체적으로 서술된 것 같은데, 이는 논제에 대한 불완전한, 반쪽짜리 이유라고 생각한다. '청소년 보호를 위해 소통의 자유를 제한하는 것이 정당하다'라는 주장을 하기 위해서는, 더 구체적으로 왜 '청소년 보호'가 '청소년 소통의 자유 보장'보다 우선시되는 가치인지에 대한 설득력 있는 이유가 두 가치 간의 비교를 통해 명시되어야 한다 그래야지만 비로소 이 주장이 온전히 성립될 수 있기 때문이다. 이에 대한 이유와 근거(사례)만 추가적으로 제시되면, 더욱 완성도 높은 논증문으로 거듭날 수 있을 것이다.

## 체크 포인트

### **구성요소**
- [ ] 논제 진술문을 찾거나 식별하기 어렵다.
- [ ] 논증 진술문을 찾거나 식별하기 어렵다.
- [ ] 증거/사례 진술문을 찾거나 식별하기 어렵다.
- [ ] 논제 재-진술문을 찾거나 식별하기 어렵다.
- [x] 논제, 논증, 증거/사례, 논제 재-진술문 각각 기능적으로 명확히 구분되어 진술되었다.

### **논제 진술의 명확성 및 표현** 
- [ ] 논제 진술문이 너무 일반적이거나 모호하여 독자가 핵심 주장을 명확히 이해하기 어렵다. 
- [ ] 논제 진술문의 핵심 용어 선택이 모호하거나, 애매하거나, 잘못되었다. 
- [x] 논제 진술문이 필자의 의도를 명확하게 전달하고 있다. 

### **논증과 증거/사례 제시** 
- [ ] 이유(Reason)가 논제와 무관하다.
- [x] 이유(Reason)가 논제를 충분히 뒷받침하지 못하고 있으며, 명확한 논리적 연결이 부족하다. 
- [ ] 증거(Example)가 구체적이지 않거나, 논제 및 이유와의 관계가 명확하지 않다. 
- [ ] 증거(Example)가 사실적 근거보다는 개인적 견해에 의존하고 있다. 
- [ ] 이유와 증거가 논리적으로 명확하게 연결되었다. 

### **논제 재-진술** 
- [ ] 논제 재-진술문이 적절하지 않거나, 단순한 반복에 그친다. 
- [x] 논제 재-진술문이 단락 도입 부분의 논제 진술을 보다 구체화, 명료화하여, 단순한 반복을 넘어서는 역할을 수행한다.
