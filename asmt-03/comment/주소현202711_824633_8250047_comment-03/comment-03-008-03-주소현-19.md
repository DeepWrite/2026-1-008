# 과제-03 단문 연습에 대한 코멘트

- 대상과제: `과제-03 단문 연습`

- 코멘트를 받는 학생: `008-19 박주은` 

---

## 코멘트

현재 제출된 OREO 논증문은 여러 구체적 사례와 근거를 통해 논의를 전개하고 있지만, OREO 구조가 명확히 보이지는 않는다는 점이 조금 아쉽다. 초반부에 조금 더 명확하게 주장(O)이 제시되고, 그 뒤에 R과 E가 따라오는 구조로 전개되면 좋을 것 같다. 또한, 두 가지 이상의 이유와 사례가 제시되고 있는데, 각각에 대한 R과 E의 경계가 좀 더 명확히 나누어지면 읽는 독자의 입장에서 글의 구조가 더 선명히 보일 것 같다. 전반적으로 이유와 근거(사례)의 구체성의 수준은 높지만, 글의 구조가 O, R, E, O로 깔끔하게 나눠지지 않는다는 점이 아쉽다. 이 점만 개선되면 훨씬 더 좋은 OREO 논증문으로 발전할 수 있을 것으로 기대된다.

### **논제 진술의 명확성 및 표현** 
- [x] 논제 진술문이 너무 일반적이거나 모호하여 독자가 핵심 주장을 명확히 이해하기 어렵다.

### **논증과 증거/사례 제시** 
- [ ] 이유(Reason)가 논제와 무관하다.
- [ ] 이유(Reason)가 논제를 충분히 뒷받침하지 못하고 있으며, 명확한 논리적 연결이 부족하다. 
- [ ] 증거(Example)가 구체적이지 않거나, 논제 및 이유와의 관계가 명확하지 않다. 
- [ ] 증거(Example)가 사실적 근거보다는 개인적 견해에 의존하고 있다. 
- [ ] 이유와 증거가 논리적으로 명확하게 연결되었다. 

### **논제 재-진술** 
- [ ] 논제 재-진술문이 적절하지 않거나, 단순한 반복에 그친다. 
- [x] 논제 재-진술문이 단락 도입 부분의 논제 진술을 보다 구체화, 명료화하여, 단순한 반복을 넘어서는 역할을 수행한다.
